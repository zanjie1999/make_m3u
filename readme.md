# make_m3u

网易云音乐m3u播放列表生成器
===================================================================================================
Python版本2.7<br>
丢到你放音乐的目录运行就好,他会在运行目录下生成你的歌单的m3u播放列表<br>
这里山灵m1播放器亲测可用<br>
